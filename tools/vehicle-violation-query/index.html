使用说明

1. 输入车牌号码：请输入完整的车牌号码，例如：京A12345

2. 选择查询地区：从下拉菜单中选择车辆注册地或违章发生地

3. 可选信息：根据需要填写车辆识别代号（VIN）或发动机号的后6位

4. 点击查询：点击"查询违章"按钮，系统将显示查询结果

⚠️ 注意事项

本工具为演示版本，使用模拟数据展示界面交互效果。实际部署时需要接入各地交通管理部门的官方API接口。请确保您有权查询该车辆的违章信息。
